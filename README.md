RSA — Практическая работа №7
============================

Криптосистемы с открытым ключом
-------------------------------

Реализация криптосистемы RSA с нуля в рамках практической работы по дисциплине «Криптографические методы защиты информации» (НИУ ВШЭ, МИЭМ).

### Цель работы

*   Реализовать криптосистему RSA полностью самостоятельно (без использования сторонних криптографических библиотек).
    
*   Освоить основные теоретико-числовые алгоритмы.
    
*   Реализовать атаку Ферма на RSA.
    
*   Продемонстрировать работу шифрования и расшифрования на практике.
    

### Функциональность

*   Генерация ключевой пары RSA (размер ключа от 16 до 2048 бит).
    
*   Шифрование текстового файла открытым ключом.
    
*   Расшифрование файла закрытым ключом.
    
*   Атака Ферма (факторизация модуля $n$).
    
*   Интерактивное консольное меню.
    
*   Работа с файлами (`*.key`, `*.txt`).
    

### Структура проекта



    /
    ├── main.py           # Основной скрипт
    ├── public.key        # Открытый ключ (e, n)
    ├── private.key       # Закрытый ключ (d, n)
    ├── input.txt         # Исходный текст
    ├── encrypted.txt     # Зашифрованный текст
    ├── decrypted.txt     # Расшифрованный текст
    ├── README.md         # Описание проекта
    └── Отчёт.docx        # Отчёт по практике

### Пример работы

**Генерация ключей (16 бит):**

*   $n = 35909$
    
*   $e = 65537$
    
*   $d = 15233$
    

**Результаты:**

1.  Успешное шифрование и расшифрование текста (включая поддержку кириллицы).
    
2.  Полное восстановление исходного файла после цикла преобразований.
    
3.  **Атака Ферма:** факторизация $n = 35909$ за 5 шагов ($p = 149$, $q = 241$).
    

### Используемые алгоритмы

*   Расширенный алгоритм Евклида.
    
*   Алгоритм быстрого возведения в степень по модулю.
    
*   Тест Ферма на простоту.
    
*   Генерация простых чисел заданного диапазона.
    
*   Метод факторизации Ферма.
